Privacy Policy

Personal Information Collection

Precise Location for App Functionality: This app collects precise location data in order to provide a location sharing service. This data is used to provide the location sharing service and is not shared with third parties. Only the last known location is stored on the server and is deleted after 24 hours. The location data is not used for any other purpose. Besides the last known location, no other location data is stored on the server. The location is stored anonymously and is not associated with any user account. Code for the server is open source and can be found at https://github.com/arbona-robin/express-loc. You can host your own server and use the app with your server.

Use of Personal Information

No personal information other than the precise location is collected or used for any purpose other than providing the app.

Information Sharing

No personal information is shared with third parties.

Changes to This Privacy Policy

This privacy policy may be updated from time to time. We will notify you of any changes by posting the new privacy policy on this page.
